### Или соберите/запустите из исходников.
Вам необходим Python 3.7 и выше.  

Из powershell  
Создайте виртуальное окружение и активируйте:
```
python -m venv .venv
.\.venv\Scripts\activate
```
Установить всё из requirements.txt:
```
pip install -r ./requirements.txt
```
Запустите main.py:
```
python main.py
```
